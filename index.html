About

In this data project I used visualization through Chart.js to show the total number of rides taken on Zagster bikes here in Bend OR from January 2018 to October 2018. I referenced a database server that contained all of the Zagster JSON data and then used the Chart.js with my HTML, JavaScript, and CSS to make everything come together how I want it to look.
